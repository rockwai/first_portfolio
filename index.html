Trump Game

HTML, CSS, JavaScript
を使い、Blackjackを作りました。下にある「Game Start」ボタンをクリックして遊んでみてください。 Blackjackのルールがよく分からない方のために以下に簡単なルールや用語の説明をさせていただきます。

Blackjackとは
ディーラーとプレイヤーの1対1の対戦形式でゲームを行い、持ち手札の数字の合計が21に近い方の勝ちというシンプルなゲームです。
BUSTEDとは
持ち手札の数字の合計が21を超えること。BUSTEDした時点で負けとなります。
HITとは
カードを引くこと。プレイヤーはBUSTEDするまでは何度でもHITすることができます。
STANDとは
HITをやめて、現状の手札で勝負をすること。
カードの数字の数え方
2~9→ その数字の通りに数えます。
10~13→ 全て10として数えます。
1→ 1または11として数えることができます。
ゲームの進行
初めにディーラーとプレイヤーに2枚のカードが配られます。プレイヤーは、表向きに表示されている ディーラーのカードを見て、HITまたはSTANDを選びます。HITをやめてSTANDすることでディーラーのターン へと移ります。ディーラーは、自身のカードの数字の合計が16以下であった場合は、17以上になるまで HITを繰り返し行います。ディーラーは、カードの数字の合計が17以上になった時点でHITをやめてSTANDします。 ディーラーがSTANDしたあとは、プレイヤーとディーラーの手札を照合して勝敗を決定します。
